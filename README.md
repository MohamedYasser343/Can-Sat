# How To Install

1. two esp32 modules
2. dht22
3. MPU9250
4. CD card
5. esp32 Cam

<br>

install the project:

```
git clone https://github.com/MohamedYasser343/Can-Sat.git
```

<br>

After changing **MAC address** in the code, upload the code.
